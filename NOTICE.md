This fork is based on Koharu 0.37.0 source history. The closest upstream
merge-base in this fork is `23f7d9fb9b4e9fcbab16ce317660cb91576ce15f` from
`mayocream/koharu`.

Fork changes include:

- macOS Apple Silicon preview DMG builds.
- Local `inpainted` and `rendered` output folders next to opened source images.
- Batch processing resilience for oversized or problematic inpainting regions.
- OpenClaw/MCP batch translation helper updates.
